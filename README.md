canteX is a web-based campus canteen ordering system. It aims to cut waiting times and make food pickup easier for students.

## Problem Statement
Students often spend a lot of time waiting in long lines at campus canteens. This causes overcrowding and inefficiency. canteX addresses this by allowing students to order food ahead of time and track their order status in real time.

## Solution
canteX allows students to:
- View menu items
- Choose multiple items and quantities
- Place orders with a preferred pickup time
- Track their order status live

Canteen staff can:
- See all incoming orders
- Update order status (Preparing, Ready, Picked Up)

## Features
- Menu-based ordering system
- Selection of multiple items and quantities
- Cart total calculation
- Real-time order status updates
- Canteen dashboard
- Firebase Firestore backend
- Mock payment options (Online, Cash on Delivery)

## Technologies Used
- HTML, CSS, JavaScript
- Google Firebase (Firestore)
- VS Code and Live Server

## How to Run the Project
1. Clone or download the repository
2. Open the folder in VS Code
3. Install the Live Server extension
4. Right-click `index.html` and select Open with Live Server
